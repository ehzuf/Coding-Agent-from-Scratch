# 从 Coding Agent 到个人助理

[基础系列](../TUTORIAL/00-overview.md) 构建了一个完整的 Coding Agent——能对话、能读写文件、能调用工具、能多 Agent 协作。但从"编程助手"到"个人助理"，还有一段距离。

本系列以 [Hermes Agent](https://github.com/nousresearch/hermes-agent) 为参考，拆解一个成熟个人助理 Agent 的核心能力，每篇配套完整代码实现。

## 这个系列讲什么

简单来说：**把一个命令行 Coding Agent，升级为可以随时通过各种平台使用的智能助理。**

整体思路是：

```
交互更聪明 → 运行更经济 → 数据更安全 → 自我进化 → 随处可达 → 主动服务
```
